# Support

Use the right channel so issues are visible and actionable.

## Bugs

Open a GitHub issue with:

- GodotMaker version
- OS and Godot version
- agent runtime, such as Claude Code or Codex
- the command or CLI workflow you ran
- relevant logs, screenshots, or `.godotmaker/*_report.json` files

## Questions and Usage Help

Use GitHub Discussions if enabled. If Discussions are not available yet, open an issue and label it as a question.

## Feature Requests

Open a feature request issue and describe:

- the game or workflow you are trying to support
- what currently blocks you
- what output or behavior you expected

## Security

Do not open public issues for security reports. Follow [SECURITY.md](SECURITY.md).
